# IP-Geolocator
Basically it will geolocate an ip.


# HOW TO USE

You need .NET SDK for this

- Windows
- Win + R
- digit cmd
- cd %USERPROFILE%\PATHTOFILE\IPGeo

If youre on powershell do:
cd "$HOME\PATHTOFILE\IPGeo"

then you'll run it

dotnet run

it will ask for an ip (i recommend scanning 8.8.8.8) if you are testing this.

- Linux
- Open your terminal by clicking it or CTRL + ALT + T
- then run the following commands
- cd ~/PATHTOFILE/IPGeo
- dotnet run
- same thing with windows, now you insert the ip and done.
